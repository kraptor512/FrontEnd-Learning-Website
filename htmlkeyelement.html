KEY ELEMENTS OF HTML

STRUCTURAL ELEMENTS:

Structural elements in HTML define the layout and organization of a web page. Key elements include:

< html >: The root element that contains all other HTML elements.

< head >: Contains metadata, links to stylesheets, and scripts.

< title >: Sets the title of the web page.

< meta >: Provides metadata like character set, viewport settings, and descriptions.

< link >: Links external resources like stylesheets.

< script >: Embeds or links JavaScript.

< body >: Contains the visible content of the webpage.




CONTENT ELEMENTS

Content elements in HTML are used to structure and display the content of a web page. Key content elements include:

Headings [< h1 > to < h6 >]: Define headings, with < h1 > being the largest and < h6 > the smallest.

Paragraphs [< p >]: Defines a paragraph.

Lists [< ul >, < ol > and < li >]: Defines Unordered (Bulletted) Lists, Ordered (Numbered) Lists and List Items respectively.

Text Formatting: There are various tags for different kind of formatting styles in HTML content like < b >/< strong > for Bold Text, < i >/< em > for Italicized Text, < u > for Underlined Text, < span > for Inline Container for styling, < br > for Line Break and < hr > for Horizontal Rule.



MEDIA ELEMENTS

Media elements in HTML are used to embed and control multimedia content on web pages. Key media elements include:

Images [< img >]: Embeds an image.

Videos [< video >]: Embeds a video with options for controls, autoplay, etc.

Audio [< audio >]: Embeds an audio file.

Embed [< iframe >]: Embeds external content like YouTube videos or Google Maps.




LINKING AND NAVIGATION

Linking and navigation in HTML are essential for creating interactive and user-friendly web pages. Key elements include:

Links [< a >]: Defines hyperlinks.

Navigation [< nav >]: Groups navigation links.




TABULAR DATA

Tabular data in HTML is used to display information in a structured, grid-like format. Key elements include:

< table >: Defines a table.

< tr >: Table row.

< td >: Table cell.

< th >: Table header cell.

< thead >: Groups Header Content.

< tbody >: Groups Body Content.

< tfoot >: Groups Footer Content.
